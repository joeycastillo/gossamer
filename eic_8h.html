void eic_configure_channel ( const uint8_t  channel,
eic_interrupt_trigger  trigger 
)

Configures an external interrupt on one of the external interrupt channels.

Note that this call does not deal with pins at all. Your external interrupt pin needs to already be configured as an input, and assigned to the EIC peripheral function. With that done, consult the pin mux table in the data sheet to see which EIC channel is associated with that pin, and pass that channel number into this function.

Parameters
channelThe external interrupt channel associated with the pin (consult the data sheet).
triggerThe condition on which you wish to trigger: rising, falling or both.
