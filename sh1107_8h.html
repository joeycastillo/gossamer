◆ sh1107_update()

void sh1107_update ( void )

Pushes the current contents of the GFX buffer to the display.

Once you have configured the GFX driver, you will draw to that buffer using the gfx draw functions. This function merely pushes the contents of that buffer to the display, and you MUST call sh1107_update to see any changes you have made.
